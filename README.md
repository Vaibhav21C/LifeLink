# LifeLink - Ambulance Tracking System

This repository contains the ambulance tracking component of the LifeLink project.

## Project Structure

- **frontend/**: React-based dashboard for real-time ambulance tracking and management.
- **backend/**: Python/FastAPI service for processing location data and routing.

## Setup Instructions

### Frontend
1. Navigate to the `frontend` directory.
2. Install dependencies: `npm install`
3. Running locally: `npm run dev`

### Backend
1. Navigate to the `backend` directory.
2. Create a virtual environment and install requirements: `pip install -r requirements.txt`
3. Run the server: `python main.py`

## Features
- Real-time ambulance tracking.
- Interactive 3D maps.
- Route optimization and visualization.
- Responsive landing page and dashboard.
